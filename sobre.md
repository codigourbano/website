O Código Urbano foi uma iniciativa de abertura de dados e produção de tecnologias livres para as cidades. Entre 2014 e 2016, em São Paulo, desenvolvemos ferramentas para facilitar a participação direta da sociedade no acompanhamento de questões políticas, econômicas, sociais e ambientais.

Nossa atuação se concentrou em três frentes: dados abertos urbanos, mapeamento colaborativo através do OpenStreetMap, e monitoramento ambiental com tecnologias livres. Ao longo desse período, construímos diversos aplicativos e plataformas — desde raspadores de dados públicos até ferramentas de mapeamento colaborativo e visualização de informações ambientais.

O projeto foi guiado por quatro princípios: transparência, licenças abertas, projetos colaborativos e o entendimento de que dados públicos são bem comum.

**Arquivado:** Este projeto está descontinuado desde 2016. O site permanece como arquivo histórico das atividades realizadas.
